# Developer Resources

Reference links for the tools and standards used in this repository.

Use this page as a starting point when you need official docs, best practices, or setup guidance.

- **Dev Containers:**
	Reproducible development environments for this project.
   	- [containers.dev](https://containers.dev)

- **EditorConfig:**
	Cross-editor formatting and whitespace consistency.
   	- [editorconfig.org](https://editorconfig.org)

- **Git Ignore:**
	Rules for excluding local or generated files from Git tracking.
   	- [git-scm.com/docs/gitignore](https://git-scm.com/docs/gitignore)
   	- [github.com/github/gitignore](https://github.com/github/gitignore)

- **Git Attributes:**
	File handling rules for diff, merge, and line-ending behavior.
   	- [git-scm.com/docs/gitattributes](https://git-scm.com/docs/gitattributes)
    - [github.com/gitattributes/gitattributes](https://github.com/gitattributes/gitattributes)

- **pre-commit:**
	Git hook framework used to run checks before commits.
   	- [pre-commit.com](https://pre-commit.com)

- **Commitizen:**
	Conventional commit message tooling used in hook workflows.
   	- [commitizen-tools.github.io/commitizen](https://commitizen-tools.github.io/commitizen)

- **prek:**
	Hook installer/runner used by this template.
   	- [pypi.org/project/prek](https://pypi.org/project/prek)

- **mise:**
	Project toolchain/version manager and task runner.
   	- [mise.jdx.dev](https://mise.jdx.dev)

- **uv:**
	Python dependency and environment workflow tool.
   	- [docs.astral.sh/uv](https://docs.astral.sh/uv)

- **Python:**
	Core language documentation.
   	- [docs.python.org/3](https://docs.python.org/3)

- **Ruff:**
	Linting and formatting.
   	- [docs.astral.sh/ruff](https://docs.astral.sh/ruff)

- **ty:**
	Static type checker.
   	- [github.com/astral-sh/ty](https://github.com/astral-sh/ty)

- **pytest:**
	Test runner and assertion framework.
   	- [docs.pytest.org](https://docs.pytest.org)

- **pytest-cov:**
	Coverage integration for pytest.
   	- [pytest-cov.readthedocs.io](https://pytest-cov.readthedocs.io)

- **ShellCheck:**
	Shell script linting and best-practice checks.
   	- [www.shellcheck.net](https://www.shellcheck.net)

- **Gitleaks:**
	Secret scanning for repositories.
   	- [github.com/gitleaks/gitleaks](https://github.com/gitleaks/gitleaks)

- **Zensical:**
	Documentation site generation.
   	- [zensical.org/docs](https://zensical.org/docs)

- **CommonMark:**
	Markdown specification reference.
   	- [commonmark.org](https://commonmark.org)

- **GitHub Actions:**
	CI/CD workflow platform used in this repository.
   	- [docs.github.com/actions](https://docs.github.com/actions)
   	- [github.com/astral-sh/setup-uv](https://github.com/astral-sh/setup-uv)

- **GitHub Copilot:**
	AI pair-programming reference docs.
   	- [docs.github.com/copilot](https://docs.github.com/copilot)

- **Model Context Protocol:**
	Open protocol for tools and model context integration.
   	- [modelcontextprotocol.io](https://modelcontextprotocol.io)

- **Visual Studio Code AI:**
	VS Code AI/Copilot feature docs.
   	- [code.visualstudio.com/docs/copilot/overview](https://code.visualstudio.com/docs/copilot/overview)

Optional AI and agent references are listed alphabetically.

## Keeping This Current

- Add tools when they are introduced in project configs or workflows.
- Remove tools that are no longer used.
- Prefer official docs over blog posts.
- Keep descriptions short and practical.
